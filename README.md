# IELTS 词根精听口语训练 V1.0

90天沉浸式雅思单词记忆 + 渐进精听 + PRE口语思维训练工具。

## 功能概览

| 功能 | 说明 |
|------|------|
| 📖 **今日学习** | 每日6个生词深度学习（词根+多语境释义+原版例句+发音） |
| 📚 **生词本** | 全部单词浏览、搜索、筛选，查看详细词卡 |
| 🎧 **精听训练** | B2难度短文，逐句复读/录音，盲听模式，语速调节 |
| 🎤 **口语练习** | PRE结构化4话题，范文跟读，录音回放复盘 |
| 🔄 **艾宾浩斯复习** | 1-2-4-7-15-30天科学复习周期 |
| 📊 **学习数据** | 连续打卡、掌握率、听力时长、周历视图 |
| ⚙ **设置** | 英音/美音切换、语速、字号、导出、重置 |

## 如何使用

### 直接运行
打开 `index.html` 即可使用。推荐用手机浏览器打开以获得最佳体验。

### 启动本地服务器（推荐，支持录音功能）
由于浏览器安全策略，录音和语音合成功能需要从 localhost 访问：

```bash
# 用 Python 启动简易服务器
python -m http.server 8000
# 然后用手机在同一 WiFi 下访问 http://你的IP:8000
```

或用 VS Code 的 Live Server 插件打开。

## 技术说明
- **语音合成**: Web Speech API（浏览器原生，免费，支持英音/美音）
- **录音**: MediaRecorder API（浏览器原生）
- **数据存储**: LocalStorage（本地存储，不清除不会丢失）
- **无需联网**: 大部分功能离线可用（TTS依赖系统语音包）

## 使用流程
1. **首页** → 点击「今日学习」开始学新词
2. 学完6个词后会自动打卡
3. **精听页** → 听短文，逐句跟读
4. **口语页** → 选话题，录自己的回答
5. 每天回**首页**查看进度，到期的词会在「复习」中提醒

## 单词数据
当前内置10个示例单词。如需导入1105词生词库，联系开发者添加。
